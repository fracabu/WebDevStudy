Javascript Learning Path

1. Fondamenti di JavaScript

Durata: 2 settimane

  • Variabili, tipi di dati, operatori
  • Strutture di controllo: if-else, switch
  • Cicli: for, while, do-while

2. Funzioni e Scope

Durata: 1 settimana

  • Dichiarazione di funzioni, espressioni di funzione
  • Arrow functions
  • Scope e closure

3. Strutture Dati e Manipolazione

Durata: 2 settimane

  • Array e metodi di array
  • Oggetti e manipolazione di oggetti
  • Map, Set

4. Programmazione Asincrona

Durata: 2 settimane

  • Callbacks
  • Promesse
  • Async/Await

5. DOM e Eventi

Durata: 2 settimane

  • Manipolazione del DOM
  • Gestione degli eventi
  • Form e validazione

6. Richieste di Rete

Durata: 1 settimana

  • AJAX
  • Fetch API
  • Axios (libreria esterna)

7. Introduzione ai Framework Front-end

Durata: 4 settimane

  • React.js (base)
  • Componenti, props e state
  • Ciclo di vita dei componenti

8. Progetto Finale

Durata: 2 settimane

Sviluppa un'applicazione web utilizzando le competenze acquisite
